# dotfiles

## Amethyst
Automatic tiling window manager for macOS.

Layouts
-------
Amethyst provides several layouts that can be used, but I only use `Row` and `Tall` for comfort.


Keyboard Shortcuts
------------------

| Shortcut | Description |
|---|---|
| `Option + Shift + r` | Relaunch Amethyst |
| `Option + Shift + Space` | Cycle layout forward |
| `Option + Shift + Left` | Shrink main pane |
| `Option + Shift + Right` | Expand main pane |
| `Option + Shift + s` | Swap focused window with main window |
| `Option + Shift + 1` | Throw focused window to screen 1 |
| `Option + Shift + 2` | Throw focused window to screen 2 |
| `Option + Shift + 3` | Throw focused window to screen 3 |
| `Option + Shift + 4` | Throw focused window to screen 4 |

## Kitty
The fast, feature-rich, GPU based terminal emulator

Keyboard Shortcuts
------------------

| Window shortcut | Description |
|---|---|
| `Command + Shift + enter` | Open new window in current directory |
| `Command + Shift + <number>` | Navigate to window `<number>`. Clockwise |
| `Command + Shift + left` | Wider window |
| `Command + Shift + right` | Narrower window |
| `Command + Shift + up` | Taller window |
| `Command + Shift + down` | Shorter window |
| `Command + Shift + d` | Close window |

| Tab shortcut | Description |
|---|---|
| `Command + Shift + t` | Open new tab in current working directory |
| `Command + Shift + [` | Go to the previous tab |
| `Command + Shift + ]` | Go to the next tab |
| `Command + Shift + w` | Close tab |

| Configuration shortcut | Description |
|---|---|
| `Command + Shift + ,` | Load configuration file |
| `Command + Shift + plus` | Increase font size |
| `Command + Shift + equals` | Decrease font size |
| `Command + Shift + zero` | Reset font size |

| Terminal shortcut | Description |
|---|---|
| `Command + c` | Copy to clipboard |
| `Command + v` | Paste form clipboard |
| `Command + up` | Scroll up |
| `Command + down` | Scroll down |
